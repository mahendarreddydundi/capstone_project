# Overleaf Import Guide

Template link provided:
- https://www.overleaf.com/read/vfprqpkynztb#a204f9

## Option 1: Paste sections into your template project
1. Open your Overleaf project from the link.
2. Open the main `.tex` file from the template.
3. Copy section content from:
   - `reports/IEEE_RESEARCH_PAPER_DRAFT.tex`
4. Replace the template title, abstract, sections, and references.

## Option 2: Upload full draft `.tex`
1. In Overleaf, click `Upload`.
2. Upload:
   - `reports/IEEE_RESEARCH_PAPER_DRAFT.tex`
3. Rename it to match your main file if needed (for example, `main.tex`).
4. Ensure class is `IEEEtran` and compile with PDFLaTeX.

## Notes
- Keep author names and affiliation as per university submission guidelines.
- Add your measured final benchmark table values if you run larger trials.
- Add figures from logs/plots before final submission.
